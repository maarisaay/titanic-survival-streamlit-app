# Titanic Survival Streamlit App

A Streamlit web application that predicts whether a Titanic passenger would survive the disaster based on passenger information such as age, sex, ticket class and fare.

The prediction is generated using a logistic regression model trained on the Titanic dataset.

## Streamlit App

https://titanic-survival-app-app-liagvzzumgz3dxtur9fugj.streamlit.app/

## Features
- Titanic survival prediction
- Survival probability estimation
- Interactive Streamlit interface
- Logistic regression model
- Passenger data input using widgets

## Installation
```bash
pip install -r requirements.txt
```

## Train Model
```bash
python train_model.py
```

## Run Streamlit App
```bash
streamlit run app/streamlit_app.py
```
